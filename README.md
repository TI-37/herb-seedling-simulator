# ハーブ苗 収益シミュレーター

兼業・副業でハーブ苗（ラベンダー／9cmポット苗）を販売する場合の、
売上・経費・利益・損益分岐点を計算するシンプルなWebアプリです。

## 特長

- **外部ライブラリ不使用**（素のHTML / CSS / JavaScriptのみ）
- 入力を変えると**自動で再計算**
- 13項目の内訳を**表形式**で表示
- **2〜5本セット**の比較表（最も粗利益が高い案を★表示）
- 販売チャネル（メルカリ／BASE／楽天／Amazon）の手数料を切り替え可能
- 金額は円単位で四捨五入

## 使い方

`index.html` をブラウザで開くだけです（サーバー不要）。

```
index.html   … 画面（入力フォーム＋結果表示）
style.css    … スタイル
script.js    … 計算ロジック（日本語コメント付き）
```

## 計算項目

総売上 / 販売手数料 / 手数料差引後売上 / 種代 / ポット代 / 培土代 /
箱代 / 送料 / 総経費 / 粗利益 / 1本あたり利益 / 損益分岐販売価格 / 損益分岐販売本数

## 拡張のヒント

- 販売チャネルは `script.js` の `CHANNELS` 配列に1行追加するだけで増やせます。
- 計算は `calcSimulation(input)` という画面非依存の純関数に集約しています。
